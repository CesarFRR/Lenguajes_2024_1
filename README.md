## Planteamiento del problema:

Un estadístico (profesional que trabaja en el campo de la estadística matemática) ha contratado a su grupo de trabajo para que le ayuden a construir un lenguaje de programación de propósito específico para el problema de dos vías de clasificación empleando los conceptos de rachas. El lenguaje debe construirse bajo el paradigma imperativo (que incluya las estructuras de secuencia, selección e iteración) y la posibilidad de definir y manejar variables, arreglos y matrices. El estadístico desea trabajar con diversas estadísticas pertenecientes al conjunto de pruebas tipo Friedman.

### Rama master

Contiene las dos carpetas de los talleres

### Rama Taller_1

En esta rama se implementó la versión inicial del compilador, que incluye únicamente el analizador léxico. Se utilizó JFlex para generar el analizador léxico a partir de las especificaciones del lenguaje.

### Rama Taller_2

En esta rama se agregó el analizador sintáctico al compilador. Se utilizó CUP para generar el analizador sintáctico a partir de la gramática del lenguaje.
